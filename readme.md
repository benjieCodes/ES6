# ES6 Syntax 

### A collection of ES6 examples that go deep into what ES6 syntax has to offer! Please take a look!

#### Includes: 

- let and const

- template literals

- destructuring objects and arrays

- fat arrow functions

- "for of" loop

- new Set and Map()

- spread operator

- generators

- Object oriented using class! (JS is not a class programming 
language!!!)

- async await

- and much more!!!!!
